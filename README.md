# Zoom VideoSDK macOS Cocoa Demo

Use of this sample app is subject to our [Terms of Use](https://explore.zoom.us/en/video-sdk-terms/).

The Zoom Video SDK for macOS lets you build custom video meeting applications with access to raw video and audio data, enabling highly-interactive, custom user experiences. This repository contains a completed sample app written in Swift with Cocoa.

## Features
This repository demonstrates the basic capabilities of the Video SDK.
- Creating/joining a session
- Starting/stopping video
- Muting/unmuting audio
